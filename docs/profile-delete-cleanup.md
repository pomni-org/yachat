# Удаление профиля: было и стало

## Было

- Подтверждение удаления открывалось через браузерный `prompt`.
- После удаления профильная папка пользователя удалялась, но часть пользовательских следов могла остаться в общем состоянии мессенджера.
- Системные сообщения с кодами подтверждения могли сохранять номер телефона удалённого профиля.
- Вложения из удалённых чатов удалялись из записей сообщений, но физические файлы в `USERS/attachments` могли остаться.
- Снимки профиля в `participantProfiles` могли сохранить старый аватар или имя, если чат по какой-то причине не был удалён целиком.

## Стало

- Подтверждение удаления открывается как модальное окно самого сайта.
- Удаление профиля чистит профильную папку вместе с аватаром из `manifest.json` и зашифрованным хранилищем.
- Сервер удаляет чаты, где профиль был участником или владельцем.
- Сервер удаляет сообщения из удалённых чатов, сохранённые сообщения профиля и системные сообщения, содержащие контакт удаляемого профиля.
- Сервер удаляет файлы вложений, связанные с удалёнными сообщениями.
- Сервер вычищает оставшиеся ссылки на профиль из `participantIds`, `participants` и `participantProfiles`.
- Локальный браузерный fallback использует ту же логику очистки для чатов, сообщений и снимков профиля.
