Условия ЯЧата

ЯЧат - web-сервис для общения. Он доступен с поддерживаемых устройств через браузер. Основное хранилище находится на стороне Vercel.

1. Сервис

Текущая версия ЯЧата работает через web-интерфейс.

2. Доступ

Страницы сервиса открываются в браузере.

3. Устройства

Поддерживаются разные устройства с современным браузером.

4. Хранилище

Основные записи находятся в базе данных Vercel.

5. Аккаунт

Аккаунт объединяет профиль и настройки.

6. Профиль

Профиль показывает выбранные пользователем сведения.

7. Ник

Ник помогает отличать аккаунты.

8. Поиск

Поиск помогает находить аккаунты внутри сервиса.

9. Чат

Чат показывает переписку выбранных участников.

10. Группа

Группа объединяет нескольких участников.

11. Название

Название помогает отличать группы и чаты.

12. Описание

Описание даёт краткий контекст профиля или группы.

13. Сообщение

Сообщение отображается в выбранном чате.

14. Ответ

Ответ связывает сообщение с предыдущей репликой.

15. Пересылка

Пересылка переносит сообщение в выбранное место интерфейса.

16. Файл

Файл добавляется через доступный элемент интерфейса.

17. QR

QR используется только для текущих функций ЯЧата.

18. Тема

Тема меняет внешний вид интерфейса.

19. Язык

Язык меняет подписи и тексты интерфейса.

20. Обновления

Сервис и документы могут обновляться.

21. Поддержка

Вопросы можно отправлять через указанные контакты.
