Политика конфиденциальности ЯЧата

Эта политика описывает текущий web-сервис ЯЧат. Сервис доступен с поддерживаемых устройств через браузер, а основные данные хранятся в серверной базе данных на стороне Vercel.

1. Сервис

ЯЧат помогает создавать аккаунт, оформлять профиль, находить пользователей, вести личные и групповые чаты, отправлять сообщения, добавлять вложения и использовать QR-сессии. Документ описывает актуальную версию сервиса.

2. Какие данные используются

  • данные входа и подтверждения аккаунта;
  • данные профиля: имя, ник, описание, аватар и дата создания;
  • данные чатов: тип, название, описание, участники, владелец группы и приглашения;
  • сообщения, ответы, пересылки, отметки редактирования и удаления;
  • вложения: имя, тип, размер, содержимое или ссылка на файл;
  • настройки интерфейса: язык, тема, страна и состояние входа;
  • технические записи, нужные для работы, безопасности и диагностики.

3. Зачем это нужно

  • для регистрации и входа;
  • для отображения профиля;
  • для работы личных чатов, групп, сообщений, вложений и приглашений;
  • для сохранения настроек;
  • для защиты аккаунта, поддержки и исправления ошибок.

4. Где хранятся данные

Основные данные ЯЧата хранятся в серверной базе данных на стороне Vercel. Браузер пользователя может хранить технические настройки и кэш интерфейса. Очистка браузера не всегда удаляет серверные данные аккаунта.

5. Кто видит данные

Данные видны пользователю, участникам выбранных чатов и группе администрирования сервиса в пределах рабочих задач. Инфраструктура Vercel используется для размещения и хранения данных сервиса.

6. Управление данными

  • Профиль можно изменить через настройки аккаунта.
  • Сообщения можно удалять через интерфейс, если действие доступно.
  • Профиль можно удалить через раздел удаления аккаунта или через обращение в поддержку.
  • После удаления часть технических записей может временно сохраняться для безопасности и стабильности сервиса.

7. Обновления политики

Политика обновляется при изменении функций, базы данных, способов хранения, внешних сервисов или требований закона. Актуальная редакция размещается на этой странице.
